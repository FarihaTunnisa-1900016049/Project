SELAMAT DATANG DI APLIKASI CORETAN MAHASISWA

Deskripsi Aplikasi

Coretan Mahasiswa adalah sebuah aplikasi web yang dapat digunakan oleh mahasiswa untuk membuat Kartu Anggota Perpustakaan. Di dalam aplikasi Coretan Mahasiswa akan memuat data diri calon anggota perpustakaan

Fitur Coretan Mahasiswa

Lihat Selengkapnya

(1) Profile, memuat contoh Kartu Tanda Anggota Perpustakaan

(2) Folder, memuat halaman untuk mengisi data diri pembuat Kartu Tanda Anggota Perpustakaan

Prosedur dan Alur Pembuatan Kartu Anggota Perpustakaan

Lihat Selengkapnya

(1) Berstatus sebagai mahasiswa aktif di Kampus

(2) Memiliki KTM (Kartu Tanda Mahasiswa)

(3) Mengisi data diri melalui aplikasi web Coretan Mahasiswa

(4) Scan Kartu Tanda Mahasiswa dalam format pdf lalu upload melalui aplikasi web Coretan Mahasiswa

(5) Pengambilan foto anggota di Perpustakaan

(6) Mengambil Kartu Tanda Anggota Perpustakaan
